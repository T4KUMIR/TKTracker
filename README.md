## 📌 Descripción

**UAT4 - T4K Tracker** es una aplicación web creada para organizar y gestionar una colección personal de videojuegos.

El objetivo del proyecto es reunir en un solo lugar todos tus juegos de diferentes plataformas como **PlayStation, Steam, Epic Games y más**, permitiendo llevar un control del progreso, logros, trofeos y estadísticas de cada jugador.

---

## 🎮 Características

✅ Biblioteca personal de videojuegos  
✅ Gestión de juegos por plataforma  
✅ Seguimiento de progreso  
✅ Registro de trofeos/logros  
✅ Organización por categorías  
✅ Interfaz moderna orientada al gaming  
✅ Diseño adaptable para PC y dispositivos móviles
